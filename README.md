## Manual  
- File > Start : Server와 연결 시도. 시간 간격 마다 데이터(기기 정보 및 센서 값) 전송.
- File > End : Server와 연결 해제
- File > Exit Program : 프로그램 종료
  
- IP 클릭 : IP주소 설정
- Port 클릭 : Port 번호 설정
